```javascript
        // –ì–µ–Ω–µ—Ä–∏—Ä—É–µ–º —Å–ª—É—á–∞–π–Ω–æ–µ —á–∏—Å–ª–æ playerC –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ (1 < C < P-1).
        // (P-2n) –¥–ª—è BigInt, Number() –¥–ª—è –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è, +2 —á—Ç–æ–±—ã –∏–∑–±–µ–∂–∞—Ç—å 0 –∏ 1.

        // –≠—Ç–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –¥–ª—è —Å—É—â–µ—Å—Ç–≤–æ–≤–∞–Ω–∏—è –æ–±—Ä–∞—Ç–Ω–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ (playerD).
        if (gcd(playerC, pMinusOne) === 1n) break; // –ï—Å–ª–∏ –≤–∑–∞–∏–º–Ω–æ –ø—Ä–æ—Å—Ç—ã, –≤—ã—Ö–æ–¥–∏–º –∏–∑ —Ü–∏–∫–ª–∞.
    }

```
